Mac Os 日历版设置步骤

  1. 1、打开iCal日历
  2. 2、点击菜单日历->订阅
  3. 3、输入一起看奥运提供的赛程订阅地址：
  4. http://2012-subscribe.baidu.cn/olympics/cal.ics
  5. 4、输入自定义日历标题及其它设置
  6. 5、查看赛程.

IOS 日历版设置

  1. 1、打开设置->邮件、通讯录、日历->添加账户->其它->添加已订阅的日历
  2. 2、填入服务器地址:http://2012-subscribe.baidu.cn/olympics/cal.ics
  3. 3、更改描述为2012奥运赛程，其它项默认，点击存卒储，完成订阅
  4. 4、打开ios日历即能同步赛程。iPad版操作类似。

google日历

  1. 1、选择其它日历->通过URL添加
  2. 2、输入订阅地址：http://2012-subscribe.baidu.cn/olympics/cal.ics，完成
